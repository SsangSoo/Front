<a href="#under">아래로 가기</a>

테이블 태그

테이블 태그는 HTML 페이지에서 표를 만들 때 사용하는 태그입니다.
하지만, 현재 div 태그를 사용하면서 사용빈도가 굉장히 줄었지만, 알아보도록 하겠습니다.
다음과 같이 사용합니다.

    <body>
        <table>

        /table>
    </body>

table 태그안에 들어갈 수 있는 요소는 많습니다만,
실제로 많이 쓰이는 태그는 다음과 같습니다.
    테이블 요소 태그
  1. tr : 표 내부의 행 태그입니다.
         table row를 의미합니다.
  2. th : 행 내부의 제목 셀 태그입니다.
         table header를 의미합니다.
  3. td : 행 내부의 일반 셀 태그입니다.
         table data를 의미합니다.

    <body>
        <table border="1">
            <tr>
                <th>Header 1</th>
                <th>Header 2</th>
            <tr/>
            <tr>
                <td>Data 1</td>
                <td>Data 1</td>
            </tr>
            <tr>
                <td>Data 2</td>
                <td>Data 2</td>
            </tr>
        </table>
    </body>


결과는 아래와 같습니다.
Header 1 Header 2
Data 1 Data 1
Data 2 Data 2

그리고 위 HTML 코드에서 보신 바와 같이 table 태그에서 속성을 설정할 수도 있습니다.
위에는 border 속성이 있는데, border 속성은 표의 테두리 두께를 지정합니다.

th 태그와 td 태그에도 속성을 지정할 수 있는데, 다음 속성을 지정할 수 있습니다.
    td와 th 요소 태그
  1. rowspan : 셀의 높이를 지정합니다.
  2. colspan : 셀의 너비를 지정합니다.


    <body>
        <table border="1">
            <tr>
                <th colspan="3">Table Data</th>
                <th rowspan="3">Table Data</th>
            </tr>

            <tr>
                <td>Table Data</td>
                <td rowspan="2">Table Data</td>
                <td>Table Data</td>
            </tr>

            <tr>
                <td>Table Data</td>
                <td>Table Data</td>
            </tr>
        </table>
    </body>

오디오 태그

오디오 태그는 웹 브라우저에서 플러그인의 도움없이 음악을 재생할 수 있게 만들어주는 HTML5 태그입니다.
HTML5에 추가된 기능으로써, 인터넷 익스플로러 8 이하에서는 사용할 수 없습니다.

사용 방법은 img 태그와 굉장히 유사합니다.
    <audio></audio>
현재 태그는 무슨 음악을 어떠한 방식으로 재생할지 정해주지 않았습니다.
속성을 활용하면 되는데, 중요한 속성들은 다음과 같습니다.


    audio 태그의 속성
  1. src : 음악 파일의 경로를 지정합니다.
  2. preload : 음악을 재생하기 전에 모두 불러올지를 지정합니다.
  3. autoplay : 음악을 자동 재생할지 지정합니다.
  4. loop : 음악을 반복할지 지정합니다.
  5. controls : 음악 재생 도구를 출력할지 지정합니다.


위의 속성들을 가지고, 음악을 재생해보도록 하겠습니다!
<audio src="001.mp3" controls="controls"></audio>

결과는 다음과 같습니다.

반면에, 다음과 같이 audio 태그에서 controls 속성이 없으면, 다음과 같이 나옵니다.
<audio src="001.mp3" ></audio>

이 글 왼쪽으로 드래그 해보시면, 왼쪽에 사진이 있음을 확인할 수 있습니다.
혹은 페이지 소스 확인하기를 통해서 확인하실 수 있습니다.

source태그

웹 브라우저 미세 버전에 따라서 audio 태그가 실행되거나, 실행되지 않을 수 있습니다.
미세 버전에 따라 실행되지 않는 문제를 해결하고자 만들어진 태그가 바로 source 태그입니다.

    <audio controls="controls">
        <source src="Kalimba.mp3" type="audio/mp3" />
        <source src="Kalimba.ogg" type="audio/ogg" />
    </audio>

결과는 다음과 같습니다.


OGG파일과 mp3파일을 사용하면 모든 브라우저에서 음악을 재생할 수 있습니다!
참고로 type 속성은 입력하지 않아도 되지만, type 속성을 입력하지 않으면,
웹 브라우저가 음악 파일을 내려받은 뒤 재생 가능한 파일인지 확인하므로, 트래픽이 낭비되기 때문에, 적어주는 것이 좋습니다.

비디오 태그

비디오 태그는 웹 페이지에서 동영상을 볼 수 있게 만들어줍니다.
HTML5 시대가 오면서 웹 표준만으로 동영상을 볼 수 있게 된 것입니다.
<참고>윈도우에 기본으로 내장되어 있는 동영상 파일은 WMV 형식인데, 웹 브라우저에서는 WMV 형식을 지원하지 않습니다. video 태그의 속성은 다음과 같습니다.

속성 이름 설명
src 비디오 파일의 경로 지정
poster 비디오 준비 중일 때의 이미지 파일 경로 지정
preload 비디오를 재생하기 전에 모두 불러올지 지정
autoplay 비디오를 자동 재생할지 지정
loop 비디오를 반복할지 지정
controls 비디오 재생 도구를 출력할지 지정
width 비디오의 너비 지정
height 비디오의 높이 지정

video 태그도 audio 태그처럼 웹 브라우저마다 지원하는 비디오 형식이 다르므로, source 태그를 사용해야 하는데요.
기본적으로 MP4 형식으로 사용하면 됩니다.

다음과 같이 사용하면 됩니다.

입력양식은 사용자에게 입력받는 공간을 의미합니다.

입력 양식은 form태그를 사용해 생성합니다.         <form>

    <form>

        <input type="submit" />
    </form>

form 태그의 속성

form 태그의 속성은 다음과 같습니다.


속성 이름 설명
action 입력 데이터의 전달 위치를 지정합니다.
method 입력 데이터의 전달 방식을 선택합니다.


method 속성에는 get 속성과 post 속성을 지정할 수 있습니다.

GET 요청입니다.

    </form>




POST 요청입니다.

    <form method="post">
        <input type="text" name="search" />
        <input type="submit" />
    </form>




GET 요청에 값을 넣고, 제출하면 url에 확인 값이 붙은 것을 볼 수 있습니다.

select 태그

select 태그는 여러 개의 목록에서 몇 가지를 선택할 수 있는 입력 양ㅇ식 요소입니다.
다음은 select 태그와 같이 사용되는 태그입니다.


태그 이름 설명
select 선택 양식을 생성합니다.
optgroup 옵션을 그룹화합니다.
option 옵션을 생성합니다.

    <select>
        <option>김밥</option>
        <option>떡볶이</option>
        <option>오뎅</option>
    </select>

fieldset 태그와 legend 태그

fieldset, legend 이 두개의 태그는 입력양식을 설명하는 태그입니다.
legend 태그는 fieldset 태그 내부에서만 사용할 수 있습니다.
다른 곳에서도 사용할 수 있지만, 아무 효과가 없습니다.


사용 예제는 다음과 같습니다.

    <form>
        <fieldset>
            <legend>입력 양식</legend>
            <table>
                <tr>
                    <td><label for="name">이름</label></td>
                    <td><input id="name" type="text" /></td>
                </tr>
                <tr>
                    <td><label for="mail">이메일</label></td>
                    <td><input id="mail"type="text" /></td>
                </tr>
            </table>
            <input type="submit">
        </fieldset>
    </form>

<a href="#up">다시 위로 가기</a>
